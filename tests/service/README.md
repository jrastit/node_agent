# Tests Service

Ce dossier contient les tests des services métiers du projet.

- **oasis/test_oasis_config_service.py** : Vérifie l'extraction des valeurs de configuration à partir de fichiers markdown.
- **oasis/test_oasis_config.py** : Vérifie la cohérence des objets de configuration Oasis.
- **oasis/test_node_info.py** : Vérifie la récupération des informations de noeud Oasis.
- **server/test_server_info.py** : Vérifie la récupération d'informations publiques sur les serveurs.

Lancez les tests avec `pytest tests/service/`.
