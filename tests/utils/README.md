# Tests Utils

Ce dossier contient les tests des utilitaires du projet.

- **test_thread.py** : Vérifie la gestion des threads et des exceptions associées.
- **test_node.py** : Fournit des fonctions utilitaires pour charger des noeuds de test.
- **api_util.py** : Fonctions utilitaires pour les appels API dans les tests.

Lancez les tests avec `pytest tests/utils/`.
